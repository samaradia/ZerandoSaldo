# Conta Digital

Projeto simples em Java para praticar conceitos básicos de Programação Orientada a Objetos (POO).

## Funcionalidades

- Criar uma conta digital
- Definir saldo inicial
- Exibir saldo atual
- Zerar o saldo da conta

## Exemplo de Saída

```text
Saldo atual: R$ 1579.42
Saldo atual: R$ 0.00
```

## Tecnologias

- Java
